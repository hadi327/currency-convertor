# Universal Currency Converter

A Python currency converter that supports global currency conversion using real-time exchange rate APIs. This project allows users to convert between international currencies with live market-based rates.

## Features

* Real-time exchange rate conversion
* Supports global currencies
* Secure API key management using `.env`
* Error handling for invalid inputs
* User-friendly terminal interface
* Continuous conversion loop

## Tech Stack

* Python
* Requests
* Python Dotenv
* Exchange Rate API

This project demonstrates API integration, secure credential management, error handling, and real-world Python application development.
